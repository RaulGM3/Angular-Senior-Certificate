Angular embraces reactive programming, where components know how to react to external events such as user clicks or data updates. When a component’s data model changes, then the component refreshes and renders its updated HTML right away.

Simple pattern to follow

  • Container components (or “smart” components): know how to get data and work with services

    • Use ChangeDetectionStrategy.Default

  • Presentation components (a.k.a. “dumb” or “lazy” components): that have to be fed with data

    Use ChangeDetectionStrategy.OnPush
