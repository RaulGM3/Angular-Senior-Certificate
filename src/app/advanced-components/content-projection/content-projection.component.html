Content Projection

Content projection is a way of passing content from a parent component to a child component so a child component is reusable in multiple cases.

You can see an example of a content projection with a class selector style in the class-selector component

  <!-- parent component -- >
  <app-class-selector>
  <h3 class="sub-title" title>
  Projected Content
  </h3>
  <p content>
  This content is projected from the parent component to the child component.
  </p>
  </app-class-selector>
  
  
  <!-- child projected component -- >
  <ng-content select="[title]"></ng-content>
  <p>This is a separation of the two projected tags</p>
  <ng-content select="[content]"></ng-content>
