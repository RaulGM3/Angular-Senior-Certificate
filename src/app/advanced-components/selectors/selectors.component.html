Selectors

Selector is a parameter in @Component decorator that creates a custom HTML tag for the component.

There are three types

  • Type selectors: app-custom-component
  • Attribute Selectors: [dropzone]:not(textarea)
  • Class Selectors: .menu-item

You can even combine them!

Attribute Selector Example

A perfect example is a Tooltip

You use it as a directive, but it adds a whole template to it.

Class Selectors Example

    div class="display-component"

This is writen within the div, but projected by ng-content to the child component
